ALL 8 ALGORITHMS

01
Uninformed Search
BREADTH-FIRST SEARCH
Explores all nodes level-by-level. Guarantees shortest path in unweighted graphs. Ideal for equal-weight evacuation grids.
O(V+E)
02
Uninformed Search
DEPTH-FIRST SEARCH
Dives deep along one path before backtracking. Supports depth limits. Useful for deep city exploration with limited memory.
O(V+E)
03
Informed Search
BEST-FIRST SEARCH
Greedy search guided by heuristic h(n) = distance + risk penalty. Fast but not always optimal. Great for time-critical evacuations.
O(V log V)
04
A* SEARCH
Informed Search
Gold standard: f(n) = g(n) + h(n). Optimal and complete with admissible heuristic. Balances cost and risk perfectly.
OPTIMAL
05
Advanced Search
AO* SEARCH
AND-OR graph search for multi-goal evacuation. Handles scenarios where multiple shelters must ALL be reached (injured + main group).
MULTI-GOAL
06
Advanced Search
HILL CLIMBING
Local optimization with random restarts. Greedily improves current path. Fast for large maps, with steepest-ascent variant.
LOCAL OPT
07
Adversarial Search
MINIMAX
Models disaster (MIN) vs. human (MAX) decisions. Prepares the safest route assuming the disaster will act against you at every step.
O(b^d)
08
Adversarial Search
ALPHA-BETA PRUNING
Minimax with intelligent branch pruning. Achieves same result as Minimax in up to O(b^(d/2)) — dramatically faster game-tree search.
PRUNED
